Weather App

HTML5 CSS3 Javascript API GitHub

For this projects, a live API was integrated to extract and share weather data. In addition to the current temperature, the users get a description, humidity and wind speed details. On the five-day forecast, users can explore the minimum and maximum temperatures. Both the icons and the background change according to the weather conditions. Vanilla Javascript techniques played a main roll on making it all come together.
